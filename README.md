## Installation

MVOnboarding is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "MVOnboarding", git: 'https://github.com/eharrison/MVOnboarding.git', tag: '0.1.0'
```
